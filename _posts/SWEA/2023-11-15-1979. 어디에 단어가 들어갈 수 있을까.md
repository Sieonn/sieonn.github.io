## 1979. 어디에 단어가 들어갈 수 있을까

N x N 크기의 단어 퍼즐을 만들려고 합니다.

주어진 퍼즐 모양에서 특정 길이 k를 가즌 단어가 들어갈 수 있는 자리의 수를 출력하는 프로그램을 만들어야 합니다.

```python
T = int(input())
for i in range(1, T+1):
    N, K = map(int, input().split())
    space = []
    for _ in range(N):
        Nn = list(map(int, input().split()))
        space.append(Nn)
    print(space)
    sol = 0
    for v in space:
        s = ""
        for x in v:
            s += str(x)
            if "1"*K in s:
                sol += 1
        print(s)        
            
                
    print(s, sol)

                count = 0
```
