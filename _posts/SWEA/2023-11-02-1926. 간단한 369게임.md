3 6 9 게임을 제작하려고 합니다. 3 6 9에 박수 대신 '-'를 출력합니니다. 30같은 경우에는 -을 출력합니다. 포함되는 개수에 맞춰 '-'를 출력합니다.

```python
N = int(input())
new = []
# rule = ("3", "6", "9")
for i in range(1, N+1):
    i = str(i)
    clap = i.count('3') + i.count('6') + i.count('9')

    if clap == 0:
        print(i, end=' ')
    else:
        print("-" * clap, end=' ')
```
