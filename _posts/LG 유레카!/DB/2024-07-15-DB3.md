## 서브커리 응용

### 테이블 생성

테이블을 복사하는 방법은 여러가지가 있다.

1. 테이블 구조와 데이터 복사

   ``` mysql
   create table 복사 테이블명
   select * from 복사할 테이블명;
   ```

   

2.  테이블 구조만 복사

   ``` mysql
   create table 복사 테이블명
   select * from 복사할 테이블명
   where 1 = 0;
   ```

   값은 빼고 구조만 복사 됩니다.

3. 테이블 구조와 원하는 데이터 복사

   ``` mysql
   create table 복사 테이블명
   select * from emp
   where deptno=30;
   ```

### 추가 수정 삭제

1. 추가

   ``` mysql
   insert into 복사 테이블명
   select * from 복사할 테이블명
   where deptno =80;
   ```

   

2. 수정

   ``` mysql
   update emp50
   set sal=sal+500
   where sal < (select avg(salary) from emp);
   ```

3. 삭제

   ``` mysql
   delete from emp
   where sal < (select avg(salary) from emp);
   ```
