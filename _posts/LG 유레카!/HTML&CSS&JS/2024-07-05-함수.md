## 함수

- 문제(Problem)
  - 현재 상태와 목표(기대) 상태 간의 차이
- 알고리즘(Algorithm)
  - 어떤 문제를 해결하기 위해 사용되는 방법(rule)과 절차(process)
- 프로그램(Program)
  - 알고리즘을 어떤 프로그램 언어의 문법에 따라 구성한 코드
- 변수(Variables)
  - 값을 저장하는 저장소 – named storage
- 함수(Function)
  - 특정한 작업을 수행하는 독립된 프로그램 단위  재활용
  - QCD(Quality, Cost, Delivery)
